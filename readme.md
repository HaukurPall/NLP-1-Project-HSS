# Setup

## KenLM

  apt-get install python3.6-gdbm python3.6-dev
  http://victor.chahuneau.fr/notes/2012/07/03/kenlm.html
  sudo apt-get install libbz2-dev
  ./bootstrap.sh
  ./bj2 -j 4
  ./bj2 install
  pip install https://github.com/kpu/kenlm/archive/master.zip --user
